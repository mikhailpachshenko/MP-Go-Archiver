| ' '                 | 11                 |
| 'n'                 | 10000              |
| 'a'                 | 011                |

| ' '                 | 11                 |

| ' '                 | 11                 |

------

Склеиваем все коды в битовую строку
```
001000000011000000111100000110000111011101001010111001000100110100101
```
------

Разбиваем на группы по 8 бит

```
00100000
00110000
00111100
00011000
01110111
01001010
11100100
01001101
00101
```

Последняя группа — 00101 → дополняем нулями до 8 бит → 00101000.
